## 🛠️ How to Use
To use these skills in your project, simply copy the desired skill folder into your local project's directory:
`your-project/.claude/skills/`

## 🌟 Available Skills

### 1. [Agentic Workflow Orchestrator](./agentic-workflow-orchestrator/SKILL.md)
**Purpose:** Turns Claude into a project manager. It breaks down complex requests into sub-tasks and manages sub-agents autonomously.
- **Best for:** Large-scale feature development and autonomous workflows.

### 2. [Performance-First Next.js](./performance-first-nextjs/SKILL.md)
**Purpose:** Enforces strict performance rules (Core Web Vitals, Image optimization, ISR) for production-ready apps.
- **Best for:** E-commerce (Perfume store), Portfolios, and SaaS.

### 3. [Enterprise CRUD Engine](./enterprise-crud-engine/SKILL.md)
**Purpose:** Standardizes the creation of complex data tables, server actions, and form handling with a focus on security and reliability.
- **Best for:** ERP Systems, Admin Dashboards, and Meter Testing Lab projects.

### 4. [Tailwind Glass UI](./tailwind-glass-ui/SKILL.md)
**Purpose:** A design-focused skill for generating modern, high-end glassmorphism components.
- **Best for:** Premium UI/UX design.
